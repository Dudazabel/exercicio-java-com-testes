### Sobre o projeto  

Este projeto foi desenvolvido em **Java**, utilizando **JDBC** para a conexão com o banco de dados MySQL.  
A proposta foi criar um **CRUD completo** para a tabela `Produto`, incluindo as operações de criação, leitura, atualização e exclusão.  
O desenvolvimento foi realizado no **IntelliJ IDEA**, com suporte do **My**
